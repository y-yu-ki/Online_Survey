# Online_Survey
システム設計1班 - オンラインアンケートサイト

## 概要 (Overview)
2026年度の「システム設計学」における、ウォーターフォールモデルによるチーム開発演習プロジェクトである。「PC Watch 不定期アンケート」をコンセプトのベースにしつつ、ユーザーが直感的にアンケートを作成・回答でき、結果を視覚的に把握できるWebアプリケーションを構築している。要件定義から内部設計までチームで協力し、検索機能や非同期処理などの実用的な拡張要件も独自に盛り込んだ。

## 技術スタック
フレームワーク（LaravelやVue、React等）は使用しない。

- OS: Ubuntu 24.04 LTS Server
- Web Server: Apache2 (2.4以降)
- Backend: PHP (8.3以降)
- Database: PostgreSQL (16以降)
- Frontend: HTML / CSS / Vanilla JS
- Libraries: jQuery, Bootstrap, Chart.js (グラフ描画), TCPDF (PDF出力)

## 主な機能 (Features)

### ユーザー機能
- 認証システム: サインアップ、サインイン機能。パスワードのハッシュ化やセッションハイジャック対策など、基本的な脆弱性対策を実装している。
- アカウント管理: ログインユーザー専用のマイページと、関連データ（作成した質問や回答、いいね等）の連鎖削除を伴う退会機能。

### アンケート作成・回答
- 柔軟な設計: 仕様はJSON形式で管理し、択一・複数選択・テキスト回答に対応している。回答の目安時間設定も可能である。
- 高度な回答UI: 回答中の進捗状況を示すプログレスバー表示や、前の回答内容に応じた条件分岐機能（Q1で「はい」と答えた人はQ2へ進む等）を実装している。
- 回答の修正: 締切前であれば、ログインユーザーは一度送信した自分の回答を修正することができる。

### 集計・分析・コミュニケーション
- リアルタイム集計: Chart.jsを使用し、結果をヒストグラムや円グラフで視覚化する。
- データ出力: 集計結果はCSV形式およびPDF形式でダウンロード可能である。
- リアクション機能: Ajaxを用いた非同期通信による、結果ページでの「コメント投稿」および「いいね」機能に対応している。
- 安全性確保: 不適切な言葉を登録させないためのNGワード制限（ハイブリッド手法）を導入している。
- 通知機能: 期限切れ間近のアンケートがある場合、ログインユーザーの画面上にサイト内通知を表示する。

## 主要ファイル構成 (Directory Structure)
- db.php / auth.php / security.php: DB接続、セッション管理、サニタイズやCSRFトークン生成を担う共通モジュール。
- index.php: トップページ。アンケート一覧の表示や、タグ・キーワード検索、並び替え機能を提供する。
- survey_form.php: アンケート作成・編集画面。入力中のブラウザバックによるデータ消失を防ぐ一時保存機能を備えている。
- question.php: アンケート回答ページ。URLのUUID実在チェックや期間外アクセスのブロックを行う。
- result.php: 結果表示ページ。
- api.php: いいねやコメント投稿、自動保存などを画面遷移なしで行うための非同期処理(Ajax)用APIである。
